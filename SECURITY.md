# Security policy

Chromet Dark is a static browser theme with no executable code in its installable package, no permissions, and no network access.

## Supported versions

Only the latest published version is supported.

## Reporting a concern

For ordinary bugs, open a GitHub issue. For a concern that should not be public, use GitHub's private vulnerability reporting feature if it is enabled for the repository.

Please include the theme version, Chrome version, operating system, and enough detail to reproduce the issue.
