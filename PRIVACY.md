# Privacy

Last updated: July 17, 2026

Chromet Dark is a visual theme for Google Chrome.

## Data collection

Chromet Dark does not collect, store, transmit, sell, or share personal information or browsing data.

## Permissions

The theme requests no Chrome permissions and no host permissions. It cannot read visited pages, browser history, bookmarks, cookies, form entries, credentials, or communications.

## Network activity

The theme contains no JavaScript, service worker, analytics, advertising, telemetry, or network requests.

## Chrome Web Store disclosure

Suggested privacy responses for the store submission:

- Single purpose: Apply the Chromet Dark visual theme to Google Chrome.
- Data collection: None.
- Data usage: None.
- Permissions justification: Not applicable. The theme requests no permissions.
- Remote code: None.

## Changes

If the project ever adds behavior beyond static theme resources, this policy and the Chrome Web Store disclosures must be updated before release.
